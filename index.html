Semantic Elements

  Semantic Elements, in the context of HTML, are words or phrases in the code with meaning. They can be understood by both the developer and the browser. Semantic markup is HTML that introduces meaning to the web page rather than just presentation. For example, a p tag indicates that the enclosed text is a paragraph. This is both semantic and representational, because people know what paragraphs are and browsers know how to display them.

Other examples of semantic elements include:

  • li
  • form
  • table
